# Chi+Ko &Tay

The world's most ambitious virtual advertising agency.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Framework Preset: **Other**
4. Output Directory: **public**
5. Click **Deploy**

Or use the CLI:

```bash
npm i -g vercel
vercel
```
